Parcial para asignatura de Tecnologías Web - UMNG

instrucciones:

* Descargar e instalar `deno`:

https://deno.land/#installation

* Ejecutar aplicación: copiar y pegar el siguiente comando en la terminal:

`deno run --allow-read --allow-write --allow-net main.js`

* Navegar a localhost:8073


Adicionalmente también se puede navegar la base de datos `database.sqlite`
usando el programa _DB Browser for SQLite_

https://sqlitebrowser.org/
